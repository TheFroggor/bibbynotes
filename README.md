simple note app I made trying to get some grips on C. 

it makes notes using nano, stores them in read-only mode in your HOME/BibnyNotes directory. Also it can open up last created note with less, show you where the notes are stored and screams "BAIIIII" upon exit.

editor can be changed in mknn.c file, code is split in one file for each of the main functions, with bibbynotes.c being the int main function.
